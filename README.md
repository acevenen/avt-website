# Ace Venen Trading — Website

Landing page and application form for Ace Venen Trading, a 1-on-1 NQ Futures coaching program.

## Pages

- **index.html** — Main landing page featuring the coaching offer, student results, and free Discord CTA
- **apply.html** — Application form collecting name, email, phone, and trading experience level

## Built With

- HTML5
- CSS3
- Vanilla JavaScript — form validation and success state on the apply page

## Features

- Mobile responsive design
- Animated hero section with P&L result cards
- Application form with client-side validation
- Connects to Discord community
- Clean dark theme with gold branding

## About

Ace Venen Trading offers a 3-month 1-on-1 NQ Futures coaching intensive. Students get 24/7 direct access to Ace, weekly calls, live trade reviews, and a system built from 10 years of trading experience. Max 30 clients at a time.

After graduating the intensive, students get access to AVT Group sessions at $147/mo.
